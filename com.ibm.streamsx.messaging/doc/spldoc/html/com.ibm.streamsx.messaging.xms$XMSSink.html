com.ibm.streamsx.messaging.xms::XMSSink

com.ibm.streamsx.messaging -> com.ibm.streamsx.messaging.xms-> XMSSink

The XMSSink operator publishes data from Streams to a WebSphere MQ Queue or Topic.

Summary

Ports
This operator has 1 input port and 1 output port.
Windowing
This operator does not accept any windowing configurations.
Parameters
This operator supports 6 parameters.

Required: connection, access

Optional: reconnectionPolicy, reconnectionBound, period, connectionDocument

Metrics
This operator reports 3 metrics.

Properties

Implementation
C++
Threading

Parameters

This operator supports 6 parameters.
reconnectionPolicy

Specifies the reconnection policy.The valid values are: NoRetry, InfiniteRetry and BoundedRetry.

Properties
  • Type: ReconnectionPolicy (InfiniteRetry, NoRetry, BoundedRetry)
  • Cardinality: 1
  • Optional: true
  • ExpressionMode: CustomLiteral

reconnectionBound

Specifies the number of successive connections that will be attempted for this operator.

Properties
  • Type: uint32
  • Cardinality: 1
  • Optional: true
  • ExpressionMode: AttributeFree

period

Specifies the time period in seconds which the operator will wait before trying to reconnect.

Properties
  • Type: float64
  • Cardinality: 1
  • Optional: true
  • ExpressionMode: AttributeFree

connection

Specifies name of the connection specification containing an XMS element. This is a mandatory parameter of the type rstring.

Properties
  • Type: rstring
  • Cardinality: 1
  • Optional: false
  • ExpressionMode: AttributeFree

access

Specifies access specification name. This is a mandatory parameter of the type rstring.

Properties
  • Type: rstring
  • Cardinality: 1
  • Optional: false
  • ExpressionMode: AttributeFree

connectionDocument

Specifies the pathname of a file containing the connection and access specifications identified by the connection and access parameters. If present, it must have exactly one value that is a String constant. If the parameter is absent, the operator will use the default location filepath ./etc/connections.xml. This is of the type rstring.

Properties
  • Type: rstring
  • Cardinality: 1
  • Optional: true
  • ExpressionMode: AttributeFree
